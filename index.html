JS and Jquery definitions

Click term and reveal the definition

  • JavaScript (JS)

    • a high-level, dynamic, untyped, and interpreted programming language.

  • Operators

    • a symbol that is used to perform an operation.

  • Variables

    • You can place data into these containers and then refer to the data simply by naming the container.

  • Variable naming conventions
    • Variables use camelCase
    • Always put spaces around operators ( = + - * / ), and after commas.
    • Always use 4 spaces for indentation of code blocks.
  • Functions

    • a function is composed of a sequence of statements called the function body.

  • Methods

    • JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition.
